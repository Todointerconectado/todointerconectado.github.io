Propiedad Transform

Nos permite mover, rotar, incrinar o modificar las dimenciones de un objeto.
Esto se puede hacer en 2D o 3D. Para lograrlo la propiedad transform utiliza funciones.

  • 2D
    • translate( ) ➡ Mueve el objeto en el eje x como en el eje y

      .muestra .translate2D:hover { transform: translate(100px, 20px); }

      transform: translate(100px, 20px);
    • rotate ( ) ➡ Gira el objeto tanto a la izquierda (left) como a la derecha (right)

      .muestra .rotate2D:hover { transform: rotate(-45deg); }

      transform: rotate(-45deg);
    • scale ( ) ➡ Permite hacer más grande o pequeño un objeto

      .muestra .scale2D:hover { transform: scale(1.5, .8); }

      transform: scale(1.5, .8);
    • skew ( ) ➡ Hace un efecto de romboide dependiendo el valor declarado

      .muestra .skew2D:hover { transform: skew(-35deg); }

      transform: skew(-35deg);
    • matrix ( ) ➡ Esta permite declarar todas las funciones anteriores en una sola linea.

      Orden de posición: matrix( scaleX(), skewY(), skewX(), scaleY(), traslateX(), traslateY() )


      .muestra .matrix2D:hover { transform: matrix(1.5, .25, .25, .8, 100, 10); }

      transform: matrix(1.5, .25, .25, .8, 100, 10);
  • 3D
    • rotateX( ) ➡ Se puede visualisar mejor con una img. Voltea la img hacia abajo o hacia arriba

      .muestra .rotateX3D:hover { transform: rotateX(180deg); }

      transform: rotateX(180deg);
    • rotateY( ) ➡ Se puede visualisar mejor con una img. Voltea la img hacia la izquierda o hacia la derecha

      .muestra .rotateX3D:hover { transform: rotateY(180deg); }

      transform: rotateY(180deg);
    • rotateZ( ) ➡ Se puede visualisar mejor con una img. Gira la img en el eje Z

      El origen es el centro del objeto. Lo cambiamos con la propiedad transform-origin dentro del padre


      .muestra .rotateZ3D:hover { transform: rotateZ(180deg); }

      transform: rotateZ(180deg);
    • rotateY( ) ➡ efecto con profundidad

      Dentro del contenedor padre va perspective:

      .muestra { perspective: 800px; height: 300px; }


      .muestra .profundidad3D:hover { transform: rotateY(180deg); }

      transform: rotateY(180deg);
